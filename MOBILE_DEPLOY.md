# Fight Simulator on iPhone

Самый быстрый путь: выложить `index.html` и `country_data.js` как статический сайт, открыть его в Safari и добавить на экран "Домой".

## Что уже готово

- `index.html` — web-версия игры
- `country_data.js` — данные игры
- `manifest.webmanifest` — чтобы сайт можно было открыть как веб-приложение

## Самый удобный вариант: GitHub Pages

1. Создай новый публичный репозиторий на GitHub.
2. Загрузи туда файлы:
   - `index.html`
   - `country_data.js`
   - `manifest.webmanifest`
3. На GitHub открой:
   - `Settings`
   - `Pages`
4. Включи Pages для ветки `main` из корня репозитория.
5. Подожди ссылку вида:
   - `https://<username>.github.io/<repo>/`
6. Открой эту ссылку на iPhone в Safari.
7. Нажми:
   - `Поделиться`
   - `На экран "Домой"`

После этого игра будет запускаться как отдельное веб-приложение.

## Что важно

- На iPhone нужно открывать именно в `Safari`.
- Если обновишь файлы в репозитории, сайт тоже обновится.
- Windows-версия через `fight_simulator.hta` остаётся рабочей отдельно.
